*Print all palindromic partitions of a string*
=

###    `Category:` `Backtrack`
###    `Level:` `MEDIUM`


Given a string s, partition s such that every string of the partition is a palindrome. Return all possible palindrome partitioning of s.

Example :
-

Input  : s = "bcc" <br><br>
Output : [ ["b", "c", "c"], <br>
          ["b", "cc"] ]

Input  : s = "geeks" <br><br>
Output : [ ["g", "e", "e", "k", "s"], <br> 
          ["g", "ee", "k", "s"] ]
